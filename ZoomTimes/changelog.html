Changelog of the Website

Welcome to the site changelog! This log mostly documents major changes to the site, be they under the hood or graphics-related, so more minor updates to the site's text may not be listed.

  • 03/28/19 - the site is now far more HTML5-compliant, using dedicated header and footer elements instead of div boxes and standalone paragraphs! This is a fairly massive organizational improvement, also allowing for far more consistent styling.
  • 09/20/18 - Revises several pages and expands Choclo and Oob's to be more accurate to several Zoomlians' present personalities.
  • 11/08/17 - Turns out I'd only tested the site on Firefox; in Google Chrome the main page looked much worse! This update fixes an issue with the changelog/blog links overlapping, and desaturates the blue color of the borders to make them easier on the eyes in Chrome and similar browsers. In addition, the code (especially the CSS styling) is much cleaner and less redundant in general, and most excitingly on the programming side, paragraph indentation is now more or less automatic, meaning I won't have to specify when to indent the beginning of a paragraph (which, for sites like these, would be pretty much all the time)!
  • 09/17/17 - No major visual or functional changes, but the site now passes the W3C validation! In layman terms, this means the site can be read more easily by any web browser without internal errors that could cause problems with the page's appearance. Additionally, this makes the general coding more professional, lets search engines find the site more readily, and gives the browsers a smoother experience to work with.
  • 06/26/17 - Another update, this time with a visual difference! The site logo and favicon are now much smoother thanks to anti-aliased recreations, and several pictures of the Zoomlians have been revised with more up to date versions. Finally, the site now has its own changelog!
  • 03/25/17 - Despite keeping the same visuals, the new Zoomlian website recieves its first major update! This update mostly features under-the-hood improved navigation with shorter URLs and a move to the more reliable GitHub, though it also includes a few minor updates to the Zoomlians' pages. Further documentation can be found here.
  • 12/03/15 - The website moves to a new location after an error with the first. Throughout the next year, minor text updates concerning the Zoomlians and their hobbies have followed. Further documentation can be found here.
  • 11/24/15 - The official website launches as a school project, based primarily around the Zoomlian family and their hobbies. Further documentation can be found here.
